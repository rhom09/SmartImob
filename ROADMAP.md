# SmartImob - Roadmap de Desenvolvimento

Este documento define as fases de evolução do SmartImob.

## 🏁 Fase 1-4: Concluídas
- Módulos base (Imóveis, Proprietários, Clientes, Contratos).
- Módulo Financeiro completo (Inadimplência, Repasses, Fluxo de Caixa).
- Refatoração: Consolidação de campos financeiros no `Property` model.

## 🚀 Fase 5: Relatórios e Dashboards (Atual)
- [ ] Dashboard Principal com métricas consolidadas.
- [ ] Gráficos interativos (Recharts) para evolução de receita e despesas.
- [ ] Exportação de relatórios em PDF/CSV.

## 🛡️ Fase 6: Segurança e Produção
- [ ] Implementação de políticas de Row Level Security (RLS) no Supabase.
- [ ] Otimização de queries para ambiente de produção.
- [ ] Configuração de ambiente de CI/CD.

## 🔮 Fase 7: Evoluções futuras
- [ ] App Mobile (React Native ou PWA).
- [ ] Integração com APIs Bancárias.
